# B42
Keyboard Driven browser, Inspired by Pentadactyl.

> Pentadactyl is dead! long live the Pentadactyl.

## How to Run
```
npm install
npm start
```

## How to Build
```
npm run build
npm run watch
npm run release
```

## Release
```
npm run release
electron-packager . b42 --platform=darwin --arch=x64 --version=0.0.1
```
